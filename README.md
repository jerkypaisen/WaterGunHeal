# WaterGunHeal
Water Gun Heal plugin for Rust
![image](https://github.com/user-attachments/assets/a1af9195-d304-485b-82a9-85bd6e2fade0)



## Permissions
```
// Heal 10
watergunheal.lv1

// Revive a fallen player.
watergunheal.lv2
```

## F1 Console Commands
```
// Give this gun to the player with the specified SteamID.
givewatergun <steamID>
```
